## 📝 Descripción

Este curso enseña técnicas prácticas, métodos estructurados y las mejores prácticas para elaborar prompts (instrucciones) sólidos y dirigir eficazmente los modelos de Inteligencia Artificial Generativa (como ChatGPT). Se exploran desde avisos zero-shot y few-shot hasta métodos avanzados como el patrón de entrevista, cadena de pensamiento (CoT) y árbol de pensamiento (ToT).
