## 📝 Descripción

El **Certificado Profesional de Ingeniería de IA Generativa de IBM** proporciona habilidades esenciales en Inteligencia Artificial Generativa, Modelos de Lenguaje de Gran Tamaño (LLMs) y Procesamiento de Lenguaje Natural (NLP). Cubre el desarrollo de aplicaciones prácticas con Python utilizando frameworks como Hugging Face, PyTorch, LangChain y RAG.
