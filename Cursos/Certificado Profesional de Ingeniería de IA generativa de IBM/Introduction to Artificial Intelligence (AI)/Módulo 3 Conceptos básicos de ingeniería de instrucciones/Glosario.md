# Glosario del Curso

¡Bienvenido! Este glosario alfabetizado contiene los nombres y definiciones de términos especializados utilizados a lo largo del curso. Identificar y describir estos términos es esencial para ayudarte a comprender los conceptos cubiertos en el curso de manera efectiva.

| Término                                                       | Definición                                                                                                                                                                            |
| :------------------------------------------------------------ | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Árbol de pensamiento                                          | Un enfoque de ingeniería de prompts que implica estructurar jerárquicamente un prompt o consulta para especificar la línea de razonamiento deseada.                                   |
| Cadena de pensamiento                                         | Un enfoque de ingeniería de prompts que implica descomponer una tarea compleja en tareas más pequeñas y manejables a través de una secuencia de prompts más sencillos.                |
| ChatGPT                                                       | Un modelo de lenguaje diseñado para proporcionar respuestas detalladas a entradas en lenguaje natural.                                                                                |
| Ciclo de retroalimentación del usuario                        | Una técnica en la que los usuarios proporcionan retroalimentación a los prompts de texto y los refinan iterativamente en función de la respuesta generada por el LLM.                 |
| Claude                                                        | Un chatbot de IA poderoso y flexible que te ayuda con tus tareas.                                                                                                                     |
| Comprensión cruzada de modalidades                            | La capacidad de los modelos de IA para conectar y razonar a través de diferentes tipos de entrada, como usar información de una imagen para responder a una pregunta basada en texto. |
| DALL-E                                                        | Modelo de texto a imagen que genera imágenes digitales a partir de descripciones en lenguaje natural.                                                                                 |
| Datos de entrada                                              | Cualquier pieza de información proporcionada como parte del prompt.                                                                                                                   |
| Difusión Estable                                              | Un modelo de texto a imagen que genera imágenes detalladas basadas en descripciones textuales.                                                                                        |
| Dust                                                          | Una herramienta de ingeniería de prompts que proporciona una interfaz de usuario web para escribir prompts y encadenarlos.                                                            |
| Enfoque de patrón de entrevista                               | Una estrategia de ingeniería de prompts que implica diseñar prompts simulando una conversación o interactuando al estilo de una entrevista.                                           |
| Enmarcado                                                     | Una técnica mediante la cual los prompts de texto guían a los LLMs para generar respuestas dentro de los límites requeridos.                                                          |
| Entorno de Desarrollo Integrado (IDE)                         | Una herramienta de software para crear y ejecutar prompts que interactúan con modelos de lenguaje.                                                                                    |
| Experiencia en dominio                                        | Una técnica en la que los prompts de texto pueden utilizar terminología específica del dominio para generar contenido en campos especializados como medicina, derecho o ingeniería.   |
| Explicabilidad                                                | Se refiere al grado en que un usuario puede entender e interpretar el proceso de toma de decisiones del modelo y las razones detrás de sus salidas generadas.                         |
| IA generativa                                                 | Un tipo de inteligencia artificial que puede crear nuevo contenido, como texto, imágenes, audio y video.                                                                              |
| IBM watsonx.ai                                                | Una plataforma de herramientas integradas para entrenar, ajustar, implementar y gestionar modelos base de manera sencilla.                                                            |
| Indicador de salida                                           | Referencias para evaluar los atributos de la salida generada por el modelo.                                                                                                           |
| Inducción comparativa                                         | Una técnica en la que se pide al modelo que evalúe múltiples salidas lado a lado para determinar cuál es mejor o más precisa.                                                         |
| Inducción de autorreflexión                                   | Una técnica en la que se instruye al modelo a revisar o criticar sus propias salidas para mejorar la precisión o creatividad. A menudo se combina con el método de eliminación.       |
| Inducción de pocos ejemplos                                   | Un método que permite el aprendizaje contextual, en el que se proporcionan demostraciones en el prompt para guiar al modelo hacia un mejor rendimiento.                               |
| Inducción de cero ejemplos                                    | Un método donde los modelos de IA generativa generan respuestas significativas a prompts sin necesidad de entrenamiento previo en esos prompts específicos.                           |
| Inducción ingenua                                             | Hacer consultas al modelo de la manera más simple posible.                                                                                                                            |
| Ingeniería de prompts                                         | El proceso de diseñar prompts efectivos para generar respuestas mejores y deseadas.                                                                                                   |
| Integración de interfaz de programación de aplicaciones (API) | El proceso de conectar diferentes sistemas de software o aplicaciones a través de sus APIs para permitir que trabajen juntos y compartan datos o funcionalidades.                     |
| Laboratorio de prompts                                        | Una herramienta que permite a los usuarios experimentar con prompts basados en diferentes modelos base y construir prompts según sus necesidades.                                     |
| LangChain                                                     | Una biblioteca de Python que proporciona funcionalidades para construir y encadenar prompts.                                                                                          |
| Método de eliminación                                         | Una técnica de prompting donde se generan múltiples salidas para la misma entrada, luego se comparan o evalúan para seleccionar o refinar la mejor respuesta.                         |
| Midjourney                                                    | Un modelo de texto a imagen que genera imágenes a partir de solicitudes en lenguaje natural.                                                                                          |
| Mitigación de sesgos                                          | Una técnica en la que los textos de entrada proporcionan instrucciones explícitamente para generar respuestas neutrales.                                                              |
| Modelos de IA generativa                                      | Modelos que pueden comprender el contexto del contenido de entrada para generar nuevo contenido. Se utilizan para la creación automatizada de contenido y comunicación interactiva.   |
| Modelos de lenguaje grande (LLMs)                             | Modelos de aprendizaje profundo entrenados con enormes cantidades de datos textuales para realizar tareas como generación de texto, resumen, traducción y análisis de sentimientos.   |
| Modelos multimodales                                          | Modelos de IA capaces de procesar y generar múltiples tipos de datos como texto, imágenes, audio y video.                                                                             |
| OpenAI Playground                                             | Una herramienta basada en la web que ayuda a experimentar y probar prompts con varios modelos de OpenAI, como GPT.                                                                    |
| Orientación contextual                                        | Una técnica mediante la cual los prompts de texto proporcionan instrucciones específicas a los modelos de lenguaje grande (LLMs) para generar salidas relevantes.                     |
| Patrón de rol/persona                                         | Un formato o estructura para construir prompts que involucran la perspectiva de un personaje o persona.                                                                               |
| Procesamiento de lenguaje natural (NLP)                       | Una rama de la inteligencia artificial que permite a las computadoras entender, manipular y generar lenguaje humano.                                                                  |
| Prompt                                                        | Instrucciones o preguntas dadas a un modelo de IA generativa para generar nuevo contenido.                                                                                            |
| PromptBase                                                    | Una mercado para comprar y vender prompts.                                                                                                                                            |
| PromptPerfect                                                 | Una herramienta utilizada para optimizar prompts para diferentes LLMs o modelos de texto a imagen.                                                                                    |
| Prompts multimodales                                          | Prompts que incluyen múltiples formas de entrada, como texto, imágenes o audio, para guiar a los modelos de IA.                                                                       |
| Scale AI                                                      | Una empresa de tecnología que se especializa en servicios de etiquetado y anotación de datos.                                                                                         |
| StableLM                                                      | Un modelo de lenguaje de código abierto basado en un conjunto de datos que contiene billones de tokens de contenido.                                                                  |

| Transformadores generativos preentrenados (GPT) | Una familia de redes neuronales que utilizan arquitectura de transformadores para crear texto o contenido similar al humano como salida. |
