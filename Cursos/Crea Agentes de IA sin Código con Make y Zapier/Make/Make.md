# Make

Es una plataforma de automatización visual que permite conectar diversas aplicaciones y servicios web para crear flujos de trabajo automatizados sin necesidad de escribir código (No-code).

## Características Principales

- **Interfaz Visual**: Se basa en un lienzo donde se arrastran y sueltan módulos para diseñar procesos complejos de forma gráfica.

- **Escenarios**: Así se denominan a las automatizaciones creadas. Estos escenarios pueden incluir múltiples pasos, filtros y lógica condicional.

- **Conectividad**: Ofrece integración con miles de aplicaciones (Google Drive, Slack, Shopify, bases de datos, APIs personalizadas, entre otras).

- **Manipulación de Datos**: Incluye funciones avanzadas para transformar datos, realizar cálculos matemáticos o procesar texto entre un paso y otro.

## Aplicación en el Repositorio

En el contexto de la IA Generativa y los Agentes, esta herramienta se utiliza frecuentemente para:

- **Orquestación**: Conectar un modelo de IA (como OpenAI o Claude) con fuentes de datos externas.

- **Automatización de Agentes**: Crear un flujo donde un correo entrante dispare un análisis por IA y el resultado se guarde automáticamente en un CRM o base de datos.

- **Procesamiento RAG**: Actuar como el puente que extrae información de un documento para enviarla al modelo de lenguaje.
