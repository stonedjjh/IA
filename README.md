# Introducción

No se trata solo de una colección de certificados, sino de un espacio de documentación activa donde traduzco conceptos técnicos a explicaciones propias y registro la resolución práctica de ejercicios

## Estructura del Repositorio

- **Cursos**: Contiene subcarpetas para cada curso que estoy realizando, con documentación detallada de cada uno, incluyendo resúmenes, ejercicios resueltos y enlaces a recursos adicionales.
