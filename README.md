# Introducción

Este repositorio centraliza mi proceso de aprendizaje y especialización en Inteligencia Artificial. No se trata solo de una colección de certificados, sino de un espacio de documentación activa donde traduzco conceptos técnicos a explicaciones propias y registro la resolución práctica de ejercicios.

## Estructura del Repositorio

- **Cursos**: Contiene subcarpetas para cada curso en realización, con documentación detallada que incluye resúmenes, ejercicios resueltos y enlaces a recursos adicionales.
- **Ejercicios**: Dentro de la carpeta de cada curso, se localiza una sección de ejercicios donde se explica el planteamiento dado y sus respectivos resultados.
